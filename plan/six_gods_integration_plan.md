# 六神排盘集成到卦象显示的计划

## 需求概述

将"六神排盘（根据日天干）"功能从独立菜单选项移除，改为在六爻起卦结果中自动显示，放在六亲之后。

## 当前状态分析

### 现有菜单结构
```
1. 标的物起卦（笔画+日时）
2. 硬币起卦
3. 数字起卦（上下卦分开输入）
4. 六神排盘（根据日天干）  ← 需要移除
0. 退出
```

### 目标菜单结构
```
1. 标的物起卦（笔画+日时）
2. 硬币起卦
3. 数字起卦（上下卦分开输入）
0. 退出
```

### 现有卦象显示顺序
1. 基本信息（名称、笔画、上下卦、动爻等）
2. 卦象信息（本卦、变卦）
3. 世应信息
4. 地支信息
5. 六亲信息  ← 六神应在此之后显示

### 目标卦象显示顺序
1. 基本信息
2. 卦象信息
3. 世应信息
4. 地支信息
5. 六亲信息
6. 六神信息  ← 新增

## 技术实现方案

### 1. 六神计算所需数据

六神排盘需要：
- **日天干**：从 `divination.py` 的 `get_day_tiangan()` 函数获取
- **六爻列表**：起卦结果中已有 `yao_list`

调用方式：
```python
from six_gods import calculate_six_gods

# 获取日天干
day_tiangan = get_day_tiangan()

# 计算六神（可选传入yao_list显示阴阳属性）
six_gods_result = calculate_six_gods(day_tiangan, yao_list=result['yao_list'])
```

### 2. 格式化输出

创建一个简洁的六神格式化函数，与现有的 `format_liu_qin_simple()` 风格一致：

```python
def format_six_gods_simple(result: dict) -> str:
    """简洁格式输出六神"""
    return f"六神: {' '.join(result['six_gods_list'])}"
```

输出示例：
```
六神: 青龙 朱雀 勾陈 螣蛇 白虎 玄武
```

在 `six_gods/utils.py` 中添加简洁格式化函数：

```python
def format_six_gods_simple(result: dict) -> str:
    """
    简洁格式输出六神（与六亲显示风格一致）
    
    Args:
        result: calculate_six_gods() 返回的结果字典
    
    Returns:
        str: 简洁格式的字符串
    
    Examples:
        >>> result = calculate_six_gods("甲")
        >>> print(format_six_gods_simple(result))
        六神: 青龙 朱雀 勾陈 螣蛇 白虎 玄武
    """
    return f"六神: {' '.join(result['six_gods_list'])}"
```
