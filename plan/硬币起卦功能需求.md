## 硬币起卦算法

### 基本规则
- 使用三枚硬币，正面=阳，反面=阴
- 抛硬币6次，每次得到一爻（从初爻到上爻）
- 每次抛三枚硬币，根据阳面（正面）数量确定爻的性质

### 爻的判定规则（正统六爻金钱卦 - 以少为尊）

| 阳面数量 | 爻的性质 | 阴阳属性 | 爻值 | 是否动爻 |
|---------|---------|---------|------|---------|
| 3个阳面 | 老阳 | 阳爻 | 1 | 是（动→变阴） |
| 2个阳面 | 少阴 | 阴爻 | 2 | 否（静） |
| 1个阳面 | 少阳 | 阳爻 | 1 | 否（静） |
| 0个阳面 | 老阴 | 阴爻 | 2 | 是（动→变阳） |

**以少为尊原则说明：**
- 2正1反：少数为反（阴），故为少阴
- 1正2反：少数为正（阳），故为少阳

### 算法伪代码

```python
def coin_toss_to_yao(coin_results):
    """
    将一次抛三枚硬币的结果转换为一爻（以少为尊原则）
    
    Args:
        coin_results: list[bool] - 三个布尔值，True=正面（阳），False=反面（阴）
    
    Returns:
        tuple: (爻值, 是否动爻, 爻类型名称)
            爻值: 1=阳爻, 2=阴爻
            是否动爻: True=动爻, False=静爻
            爻类型名称: "老阳"/"少阳"/"少阴"/"老阴"
    """
    heads_count = sum(coin_results)  # 阳面（正面）数量
    
    if heads_count == 3:
        return (1, True, "老阳")   # 老阳：阳爻动
    elif heads_count == 2:
        return (2, False, "少阴")  # 少阴：阴爻静（以少为尊，2正1反，反为少，故阴）
    elif heads_count == 1:
        return (1, False, "少阳")  # 少阳：阳爻静（以少为尊，1正2反，正为少，故阳）
    else:  # heads_count == 0
        return (2, True, "老阴")   # 老阴：阴爻动
```

### 动爻判定
- 老阳（3个阳面）和老阴（0个阳面）为动爻
- 从六爻中找出所有动爻的位置
