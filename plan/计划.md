# 六爻起卦系统重构计划

## 项目概述
重构六爻起卦系统，实现三种新的起卦方式，替换原有的起卦方式。

## 起卦方式（3种）

### 1. 标的物起卦（笔画+日时）
**算法流程：**
1. 用户输入标的物名称（简体）
2. 简体转繁体
3. 计算繁体字笔画数
4. 获取农历年月日时
5. 计算：
   - 上卦 = (笔画数 + 农历年 + 农历月 + 农历日) % 8，余数0取8
   - 下卦 = (笔画数 + 农历时) % 8，余数0取8
   - 动爻 = (笔画数 + 农历年 + 农历月 + 农历日 + 农历时) % 6，余数0取6

**八卦对应：**
- 1=乾、2=兑、3=离、4=震、5=巽、6=坎、7=艮、8=坤

**技术需求：**
- 简体转繁体库（如opencc）
- 繁体字笔画字典
- 农历日期转换算法

---

### 2. 随机起卦
**算法流程：**
1. 生成6个随机数（0-3）
2. 直接作为六爻值

**爻值定义：**
- 0 = 少阴（阴爻，静）
- 1 = 少阳（阳爻，静）
- 2 = 老阴（阴爻，动→变阳）
- 3 = 老阳（阳爻，动→变阴）

---

### 3. 数字起卦（上下卦分开输入）
**界面：**
```
上卦数字: [输入框]
下卦数字: [输入框]
```

**算法流程：**
1. 上卦数字 % 8 → 余数对应八卦（余数0取8）
2. 下卦数字 % 8 → 余数对应八卦（余数0取8）
3. 动爻 = (上卦数字 + 下卦数字) % 6，余数0取6

**示例：**
- 输入：上卦=9，下卦=9
- 上卦：9 % 8 = 1 → 乾卦
- 下卦：9 % 8 = 1 → 乾卦
- 动爻：(9+9) % 6 = 18 % 6 = 0 → 取6 → 6爻动
- 结果：乾为天卦，6爻动

## 从八卦到六爻

上下卦组合成六爻：
- 上卦3爻（第4、5、6爻）
- 下卦3爻（第1、2、3爻）

**示例：**
- 上卦=乾(111)，下卦=坤(000)
- 六爻 = 111000（从下到上：000111）
- 对应：天地否卦
